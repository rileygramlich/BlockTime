# BlockTime

“The weekly planner that schedules around your energy, not just your deadlines.”

## Try Now

[BlockTime](https://rileygramlich.github.io/BlockTime/)

## Description and Problem Statement

- Students often fail to plan their weeks effectively because they treat every hour as equal
- They don't account for fluctuating energy levels which lead to burnout or procrastination
- The solution to this is in the form of an app called BlockTime is a smart weekly planner that generates "work blocks" based not just on due dates and priorities, but on the day-to-day energy levels of the use which are low, medium, or high
- It automates the decision of what to do and when, guaranteeing that heavy tasks are matched with high energy days

## Key Features

- **Quick Input/Output:** Simple text-based input for easy copying and a clear 7 day schedule output
- **Schedule Management:** Options to set weekly availability, and view daily work block options
- **Capacity Alerts:** Visual indicators if the user's available hours are insufficient to complete the assigned tasks
- **Smart Constraints:** Limits work to a maximum of 2 major tasks per day to prevent cognitive overload and keep students on track
- **Energy-Based Scheduling:** Automatically adjusts task allocation based on daily energy input such as how low energy days focus on lighter tasks and High Energy days tackle heavy projects

## Getting Started

Below are the steps that are used to run the app:

1. **Using Git Bash, clone the repository:** git clone https://github.com/rileygramlich/BlockTime.git
2. **Using Git Bash, Navigate to Directory:** cd BlockTime
3. **Launch:** If the project is Static (HTML/JS), open index.html directly in your browser. If the project uses React/Node, run npm install followed by npm start
